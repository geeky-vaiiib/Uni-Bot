# SIT Library - Complete Guide

## Overview
The Central Library at SIT is one of the largest engineering college libraries in Karnataka, serving as a knowledge hub for students and faculty.

**Location**: Central Library Block, Near Administrative Building
**Total Area**: 2500+ sq. meters
**Seating Capacity**: 500+ students

## OPAC (Online Public Access Catalog)

### How to Search
1. Visit library.sit.ac.in
2. Click on OPAC
3. Search by: Title / Author / Subject / Accession No.
4. Check availability and location
5. Note call number for finding book

### Book Location Code
```
Example: 005.133 JAV P2-S3-R5
005.133 = Subject (Java Programming)
JAV = Author code
P2 = Second Floor
S3 = Stack 3
R5 = Rack 5
```

## Library Rules

### General Rules
1. Maintain silence in all areas
2. Show ID card at entry
3. Do not bring bags inside (use lockers)
4. Mobile phones on silent mode
5. No food or drinks
6. Handle books carefully
7. Do not mark or tear pages
8. Return books on time

### Borrowing Rules
1. Check book condition before issue
2. Report damage immediately
3. Books are non-transferable
4. Return before exams
5. Clear dues before semester end

### Penalties

| Violation | Penalty |
|-----------|---------|
| Overdue books | ₹1/day/book |
| Lost book | Cost + 50% fine |
| Damaged book | Assessed replacement |
| Marking pages | ₹100 + repair |
| Misconduct | Library suspension |

## FAQs

**Q: Can I access e-resources from home?**
A: Yes, using VPN or RemoteXs credentials from library.

**Q: How many books can I issue at a time?**
A: UG: 4 books, PG: 6 books, Faculty: 10 books.

**Q: What if I lose a book?**
A: Pay original cost + 50% fine, or provide same edition.

**Q: Can I extend book return date?**
A: Yes, once for 7 days if no reservation.

**Q: Is 24x7 access available?**
A: Only during exams with prior permission.
